# 데이터 프레임 : 2차원으로 구성되고 시리즈가 모여 만들어짐
```python
- 생성: df = pd.DataFrame([ ['a','b','c'],['a','a','g'],['a','g'] ])
    - 값의 갯수 확인: df.size
    - 행,열  갯수 확인: df.shape
- 딕셔너리로 생성 하면 키값이 컬럼명이 됨 value를 리스트로 입력해 row의 값이됨

- 시리즈들로 데이터 프레임 생성
    - a = pd.Series([100, 200, 300], ['a', 'b', 'd'])
       b = pd.Series([101, 201, 301], ['a', 'b', 'k'])
       c = pd.Series([110, 210, 310], ['a', 'b', 'c'])
    -  pd.DataFrame([a,b,c])

- 열추가: df[추가 할 컬럼명] = value
- 열 삭제: del df[삭제할 컬럼명] / 완전삭제됨
- 열 인덱싱 : 키값으로 추출함
    - 열추출: df['컬럼명'], df . 컬럼명, df[ ['컴럼명'] ] / df형태로 추출
    - 복수열 추출: df[ ['컬럼명1','컬럼명2'] ]
    - 개별요소 접근: df[열][행]

- 데이터 프레임 인덱서 : loc, iloc + 기존 인덱싱(추출)방식과는 다름
####생성법
    - df.loc[행제목,열제목] : 라벨값 기반의 2차원 인덱싱
        - 행 우선 인덱서: df[행인덱싱 값]
        - 컬럼 인덱싱 시 슬라이싱을 사용: df.loc[ [ : ,'c' ] ]
        - loc는 인덱싱 슬라이싱 시 1:3일 경우 1부터 3까지 출력함

    - df.iloc[행번호, 열번호] : 순서를 나타내는 정수기반의 2차원 인덱싱
        - 인덱스값은 0부터 시작함
        - 
    - boolean selection으로 행 선택하기 : 비교 연산자를 사용하면 boolean 값으로 각 요소를 대입하여 불리언시리즈형으로 반환 (ex: df.A > 15 )
        -데이터프레임의 비교연산자 df[ df.A > 15 ]: 값이 참인 행이 반환됨 
``````
